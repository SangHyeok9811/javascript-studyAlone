1. 변수 - 변하는 값을 저장하는 저장공간(그릇)
        2. 형식
            var 변수명 = 값;

        3. 변수명 규칙
            1. 영문자, 숫자, $ 혼용할 수 있다
            2. 대소문자 구별한다. a = A 다르다
            3. 한글, 예약어(int, String 같은 사용하는 언어) 특수문자 사용할수 없다.
            4. 첫글자 반드시 숫자를 제외한 영어 (주로 소문자로 시작)
            
            var 그릇1 = 밥 => 밥그릇
            var 그릇2 = 국 => 국그릇
            var 그릇3 = 콜라 -> 음료수 그릇
            var 선언문
                그릇1 => 변수
